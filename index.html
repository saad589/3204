Course Description¶

This lab course is designed to help the students to get knowledge on modeling and simulation using MATLAB and LabVIEW softwares. Students should have intermediate to advanced level of understanding of linear algebra, numeiral methods and high-level programming.

Learning Outcomes¶

On completion of the course, students are able to:

  • Develop an understanding of modeling and simulation concepts used in engineering.

  • Learn how to analyze and interpret results, and make conclusions based on the analysis.

  • Gain problem-solving skills through experiments and projects that involve the use of modeling and simulation techniques.

  • Understand the fundamentals of data acquisition, processing, and analysis using virtual instruments.

Groups¶

Tasks are (mostly) individually assigned with exceptions; please find the group distribution here: Groups

Reporting Instructions¶

A report is to be submitted physically within 1/2 weeks (referring to the due date) of the assignment at the latest; late submissions will not be accepted. Existing plagiarism rules will apply.

Students are advised to use the following format for the report:

  • Cover page

  • Theory

  • Algorithm

  • MATLAB code

  • MATLAB output

  • Result

  • Discussion of result
